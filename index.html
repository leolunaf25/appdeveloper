Sistema de autenticación en Android utilizando Firebase Authentication.

Desarrollé un sistema de autenticación en Android que permite registrarse y acceder con correo y contraseña, además de iniciar sesión con Google. Usé Kotlin como lenguaje principal e integré Firebase Authentication para gestionar el acceso de manera segura.
También implementé Google Sign-In para facilitar un inicio de sesión rápido. Siguiendo el patrón MVVM, estructuré el proyecto de forma organizada y escalable. Para la interfaz, utilicé Material Design, asegurando una experiencia intuitiva y moderna.

Shop Explorer, del popular juego en línea.

Desarrollé una aplicación que consume datos de una API pública para mostrar la tienda del día, los aspectos visuales disponibles, los detalles de cada diseño y el mapa actual con ubicaciones clave.
Para la comunicación con la API, utilicé Retrofit, asegurando un consumo de datos eficiente. Implementé Picasso para la carga y gestión optimizada de imágenes, y empleé corrutinas para manejar procesos asíncronos, garantizando una experiencia fluida.
Además, estructuré el proyecto siguiendo MVVM y Clean Architecture, facilitando la escalabilidad y el mantenimiento del código.

Videojuegos

He desarrollado varios videojuegos con mecánicas únicas, aprovechando la versatilidad de Unity Engine para crear experiencias interactivas y dinámicas.
Utilicé C# como lenguaje principal para la lógica del juego y la gestión de eventos, junto con el Physics Engine de Unity para simular movimientos y colisiones de manera realista.
Además, diseñé interfaces intuitivas y optimizadas para el jugador, aplicando principios de UI/UX Design para mejorar la experiencia de usuario.
